# Smart City Intelligence Dashboard

An enterprise-grade smart city command center and municipal intelligence dashboard featuring transit telemetry metrics, environmental sensor tracking, emergency responder dispatch controls, and live traffic log terminals. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Municipal Header**: Platform logos, operational status listings, and alert warnings.
- **Urban Indicator Cards**: Quad layouts detailing average transit flows, air quality (AQI) ratings, electrical grid draw stats, and pending incident counts.
- **Transit Telemetry Table**: Table listing subway routes, highway load statuses, delay estimates, and transit delays.
- **Emergency Action Panel**: Command controls dispatching emergency responders or adjusting power grids.
- **Grid Stream Console**: Monospaced terminal outputting active dispatcher logs.

## 🎨 Theme Details

- **Palette**: Dark Urban (`#090c10`), Deep Slate (`#151922`), Success Green (`#10b981`), Cyan Accent (`#0ea5e9`), Warning Amber (`#f59e0b`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft entrance sweeps, warning blinks, and status pulses.
