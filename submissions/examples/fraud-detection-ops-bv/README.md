# Fraud Detection Operations Center

An enterprise-grade fraud intelligence and detection dashboard featuring transaction alert tables, risk scoring indexes, verification controllers, and live threat detection terminals. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Fraud Monitor Header**: Logo layouts, security metrics, and active threat levels.
- **Threat Index Cards**: Quad layout metrics detailing flagged transaction totals, safe percentage ratios, saved chargeback metrics, and average risk indices.
- **Transaction Alert Table**: Tables detailing flagged transactions, dollar values, risk points, geolocation warning triggers, and mitigation stages.
- **Quarantine Action Console**: Playbook controls triggering manual validation actions or payment clearances.
- **Detection Log Terminal**: Monospaced logging terminal showing active engine decisions.

## 🎨 Theme Details

- **Palette**: Dark Cyber (`#090a12`), Surface Navy (`#141727`), Warning Amber (`#f59e0b`), Threat Red (`#ef4444`), Status Green (`#10b981`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Warning sweeps, scale pulses, and interactive transitions.
