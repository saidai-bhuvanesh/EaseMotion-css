Navigation Bar Text Wrapping & Alignment Fix

This page demonstrates the issue where multi-word navigation links wrap on small screens, causing uneven heights and baseline misalignment, alongside the fix using white-space: nowrap and Flexbox centering.

❌ Wrapped Links & Misaligned Baseline (Before Fix)

In a narrow header container, text links like "Getting Started" and "Animation Preview" wrap onto multiple lines. Because list items (li) are block-level (no flexbox centering) and links are allowed to wrap, the heights are inconsistent and elements are misaligned.
