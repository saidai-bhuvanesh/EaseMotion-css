# Digital Twin Monitoring Platform

An enterprise-grade physical asset simulation and digital twin monitoring dashboard featuring sensor status registries, latency counters, operational calibration controllers, and live physical engine log streams. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Twin Header**: Twin logos, synchronization indicators, and active physics status badges.
- **Physical Telemetry Cards**: Quad layouts detailing asset health indexes, sync latency, temperature metrics, and warning counts.
- **Sensor Registry Table**: Tables showing vibration and thermal sensor values, tolerance limits, and operational statuses.
- **Calibration Action Console**: Strategical controllers enabling sensor alignments and physics recalculations.
- **Engine Stream Terminal**: Monospaced terminal logging physical updates.

## 🎨 Theme Details

- **Palette**: Space Black (`#07090e`), Slate Navy (`#101524`), Cyan Accent (`#0ea5e9`), Highlight Purple (`#8b5cf6`), Threat Red (`#ef4444`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft sweeps, status beacon glows, and transition buttons.
