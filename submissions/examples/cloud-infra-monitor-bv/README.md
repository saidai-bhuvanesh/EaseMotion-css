# Cloud Infrastructure Monitoring Hub

An enterprise-grade cloud operations and infrastructure monitoring dashboard featuring resource metrics, cluster node status tables, scaling controls, and live container logging. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **SLA Header**: Core brand logos, platform active statuses, and SLA uptime score values.
- **Resource Metrics Cards**: Quad layouts detailing average CPU loads, allocated memory ratios, throughput networks, and alert queues.
- **Node Cluster Table**: Grid listing region classifications, usage ratios, and node IDs.
- **Replica Action Panel**: Controls triggering automated container scale additions.
- **Cluster Stream Shell**: Monospaced logging terminal showing infrastructure events.

## 🎨 Theme Details

- **Palette**: Dark Navy (`#090d16`), Slate Surface (`#131c2c`), Success Emerald (`#10b981`), Cyan Accent (`#0ea5e9`), Warning Amber (`#f59e0b`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Dynamic entrance sweeps, scale pulses, and interactive transitions.
