# Interactive Product Showcase Hub

An ultra-premium product discovery showcase platform featuring mesh background gradients, interactive layout tabs, onboarding stepper timelines, analytics telemetry cards, and FAQ accordions. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Product Hero**: Rich header with dynamic radial gradient meshes and action buttons.
- **Features Grid**: Three card layouts showing product benefits.
- **Workflow Tab Area**: Interactive switch layout simulating active dev configurations.
- **Live Telemetry Cards**: Numeric gauges tracking uptime, latency, and throughput stats.
- **Onboarding Stepper**: Connecting sequence outlining starting processes.
- **Partner Logo Array**: Grid display listing supported integration vendors.
- **Roadmap Timeline**: Roadmap list showing prospective releases.
- **FAQ Accordion Details**: Expandable answer segments detailing deployment options and overheads.

## 🎨 Theme Details

- **Palette**: Dark Black (`#0a0a0f`), Dark Blue Surface (`#12121e`), Primary Blue (`#3b82f6`), Accent Violet (`#a855f7`), Warm Orange (`#f97316`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft floating header effects, vertical hover cards, and timeline transitions.
