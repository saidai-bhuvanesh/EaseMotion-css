# Enterprise Testimonial Intelligence Platform

An ultra-premium customer success dashboard and reputation intelligence center showing CSAT score dials, case study matrix lists, onboarding timelines, and dynamic trust badges. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### 16 UI Sections

| #   | Section                           | Description                                                                     |
| --- | --------------------------------- | ------------------------------------------------------------------------------- |
| 1   | **Header**                        | Platform branding with average brand trust score indicators                     |
| 2   | **Trust Telemetry Dashboard**     | Performance cards tracking NPS, CSAT, review counts, and average star ratings   |
| 3   | **Enterprise Success Stories**    | Customer quote layouts displaying photos, tags, names, and titles               |
| 4   | **Video Advocates**               | Video placeholder panels with hover play trigger buttons                        |
| 5   | **Advocacy Engagement Analytics** | Telemetry tracking social impressions, shares, and demo conversions             |
| 6   | **Sentiment Monitoring**          | Positive/Neutral sentiment percentage tracks with visual completion bars        |
| 7   | **Reputation Intelligence**       | Cards displaying recent industry press mentions and index standings             |
| 8   | **Review Source Aggregator**      | Trust aggregator indicators showing ratings across G2, Trustpilot, and Capterra |
| 9   | **Success Onboarding Pipeline**   | Three-step horizontal pipeline detailing advocacy steps                         |
| 10  | **Case Study Matrix**             | Industry tagged blocks detailing case study details and solutions               |
| 11  | **ROI Impact Metrics**            | Impact stats detailing speedups, budgets saved, and audit values                |
| 12  | **Social Proof Wall**             | Micro-card tweets and posts showing user mentions                               |
| 13  | **Advocacy Leaderboard**          | Detailed table scoring high-value advocates and referrers                       |
| 14  | **Testimonial Request Pipeline**  | Categorized kanban-style stages showing customer status                         |
| 15  | **Story Release Calendar**        | Dynamic weekday grid mapping testimonial releases                               |
| 16  | **Dynamic Trust Badges**          | Secure badge generator panel showing code integrations                          |

## 🎨 Theme Details

- **Palette**: Dark Surface (`#1a1a1a`), Deep Grey (`#222222`), Gold Accent (`#eab308`), Success Teal (`#14b8a6`), Love Rose (`#fb7185`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Entrance slide-ups, trust pulse logos, star rating scale lifts, and hover lifts.
