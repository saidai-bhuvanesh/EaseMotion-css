# Healthcare Intelligence Operations Platform

An enterprise-grade healthcare analytics and operational command platform featuring patient intake registries, ICU capacity dials, doctor timeline rosters, and live clinical console streams. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Clinical Header**: Command brand logo, facility active indicators, and health status values.
- **Resource Allocation Cards**: Quad layouts detailing occupied ICU percentages, emergency wait times, active staff totals, and pending critical alerts.
- **Patient Intake Table**: Registry tables listing priority indicators, assigned wards, attending clinicians, and intake IDs.
- **Roster Action Panel**: Command controls dispatching on-call clinicians or compiling audits.
- **Clinical Stream Console**: Monospaced terminal outputting active hospital metrics.

## 🎨 Theme Details

- **Palette**: Dark Clean (`#080c14`), Deep Slate (`#111a2e`), Success Teal (`#14b8a6`), Cyan Accent (`#0ea5e9`), Threat Red (`#ef4444`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft entrances, capacity gauge blinks, and status pulses.
